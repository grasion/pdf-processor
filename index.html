🎯 주요 기능

✂️ PDF 분할

PDF 파일을 페이지 범위, 개별 페이지, 또는 페이지 단위로 분할할 수 있습니다.

🔗 PDF 병합

여러 PDF 파일과 이미지를 하나의 PDF로 병합할 수 있습니다.

🖼️ 이미지 변환

PDF를 PNG, JPG, BMP, TIFF 이미지로 변환할 수 있습니다.

📖 사용 방법

PDF 분할하기

PDF 파일 선택
좌측 상단의 "PDF 파일 선택" 버튼을 클릭하거나 파일을 드래그하여 추가
분할 옵션 선택
페이지 범위, 개별 페이지, 또는 페이지 단위 중 선택
출력 폴더 지정
분할된 파일이 저장될 폴더를 선택 (기본: 원본 파일 위치)
분할 실행
"PDF 분할 실행" 버튼을 클릭하여 처리

PDF 병합하기

파일 추가
병합할 PDF 및 이미지 파일을 드래그하거나 "추가" 버튼으로 선택
순서 조정
↑↓ 버튼으로 파일 순서를 조정
저장 위치 설정
파일명과 저장 위치를 지정
병합 실행
"PDF 병합 실행" 버튼을 클릭하여 처리

⚠️ PDF 변환 유의사항

처리 전 확인사항

  • 암호화된 PDF는 먼저 암호를 해제해야 합니다
  • 대용량 PDF 파일은 처리 시간이 오래 걸릴 수 있습니다
  • 이미지 변환 시 DPI가 높을수록 파일 크기가 커집니다
  • 병합 시 이미지는 자동으로 PDF 페이지 크기에 맞춰집니다
  • 처리 중에는 프로그램을 종료하지 마세요

권장 설정

  • 이미지 변환: 일반 문서는 150 DPI, 고품질은 300 DPI 권장
  • 병합 시 페이지 크기: A4 세로 방향이 가장 범용적
  • 파일명 패턴: 날짜와 페이지 번호를 포함하면 관리가 편리
  • 출력 폴더: 바탕화면이나 문서 폴더 사용 권장

📋 지원 형식

입력 형식

PDF: 모든 버전
이미지: PNG, JPG, JPEG, BMP, TIFF, GIF

출력 형식

PDF: 표준 PDF 1.4+
이미지: PNG, JPG, BMP, TIFF

해상도 옵션

72, 96, 150, 200, 300, 600 DPI
